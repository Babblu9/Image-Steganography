# Image-Steganograpgy

Image steganography is the process of concealing the data in a image.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to use the code

```bash
pip install numpy
```
```bash
pip install cv2
```
```bash
pip install types
```
